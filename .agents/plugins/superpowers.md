# Plugin: Superpowers
Estructuración de flujos de desarrollo para el trabajo de software (planificación, lluvia de ideas y ejecución).

## Comandos:
- /planificar: Genera un desglose detallado de los pasos técnicos requeridos.
- /idear: Propone alternativas de arquitectura limpias y eficientes.
- /ejecutar: Escribe bloques de código limpios y documentados.
