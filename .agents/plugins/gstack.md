# Plugin: Gstack
Define roles de ingeniería especializados para guiar la simulación del ciclo de vida del software.

## Roles Definidos:
- QA (Quality Assurance): Validador y diseñador de pruebas de código.
- Software Architect: Diseñador de bases de datos y APIs.
- Technical Writer: Redactor de documentos técnicos (por ejemplo, los archivos de Word del estudiante).
