# Skill: Frontend Design (Alta Fidelidad Estética)
Optimiza la generación de interfaces de usuario rompiendo con patrones algorítmicos estándar.

## Directrices de Diseño:
1. Usar paletas de colores HSL armoniosas y personalizadas (evitar colores puros primarios).
2. Incorporar gradientes sutiles y efectos de vidrio (glassmorphic) cuando sea apropiado.
3. Asegurar tipografía de alta legibilidad y jerarquía visual clara.
4. Implementar micro-transacciones en elementos de hover y transiciones suaves.
